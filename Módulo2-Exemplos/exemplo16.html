Pop, Push, Shift, Unshift, Sort e Reverse

pop()

O método pop() remove o último elemento do Array

shift()

O método shift() remove o primeiro elemento do Array

push()

O método push() adiciona um elemento no Array

Unshift()

O método unshift() adiciona no início um novo elemento no Array

Sort() e Reverse()

O método sort() ordena um Array

O método reverse() classifica decrescente um Array
